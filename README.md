# AI Productivity Hacker - Kamus Coding & Visual Explainer

Repositori ini berisi kumpulan file HTML interaktif yang dibuat untuk memudahkan pemahaman konsep-konsep *software engineering*, arsitektur web (seperti Next.js), dan kamus syntax JavaScript/React menggunakan pendekatan analogi sederhana. Sangat cocok sebagai bahan pembelajaran atau materi konten edukasi.

## Struktur Direktori

### 1. `/diagrams`
Kumpulan "Kamus Visual" dan diagram arsitektur interaktif yang bisa dibuka langsung di browser:
- `coding-analogies.html` - Penjelasan Fundamental Coding (Variabel, Fungsi, Algoritma) dengan analogi sehari-hari.
- `javascript-restaurant.html` - Penjelasan Arsitektur Fullstack / Next.js (Frontend, Backend, Database) menggunakan perumpamaan "Restoran".
- `top-20-syntax.html` - Cheat Sheet Top 20 logika & simbol (*Syntax*) paling sering muncul dalam kode hasil *generate* AI.
- `kamus-syntax-lengkap.html` - Kamus besar interaktif yang mengkategorikan *syntax* alien ke dalam logika singkat, pengolahan data, dan khusus React/Next.js.
- `syntax-cheat-sheet.html` - Ringkasan singkat 3 syntax esensial: `fetch`, `async/await`, dan *arrow function*.
- `fase0-latihan.html` - Diagram / file referensi tambahan.

### 2. File HTML Lainnya
Beberapa eksperimen mandiri dan UI yang tersimpan di direktori utama:
- `kalkulator-usia-ios.html`
- `Kaskita.html`

## Cara Penggunaan
Cukup *clone* repositori ini atau *download* langsung, lalu klik dua kali pada file `.html` mana pun untuk membukanya di browser favoritmu (Chrome, Safari, Edge, dll). Tidak perlu *server* atau instalasi tambahan!
