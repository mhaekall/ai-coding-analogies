Top 20 Cheat Sheet "Bahasa" Mesin

Ini adalah 20 kata kunci / simbol yang mencakup 90% dari keseluruhan kode Next.js/React buatan AI yang akan kamu temui.

Syntax / Simbol Apa namanya? Analogi / Cara Membacanya
Fase 1: Komunikasi & Pengiriman (Backend)
fetch()
Mengambil/Mengirim Data
 "Panggil kurir untuk pergi ke alamat tersebut."
async / await
Menunggu Proses
 "Tolong diam dan tunggu sampai kurirnya selesai, baru lanjut ke baris bawahnya."
JSON.stringify()
Membungkus Paket
 "Bungkus dan lakban kotak paket ini sebelum dikirim ke internet."
JSON.parse()
Membuka Paket
 "Buka lakban paket yang baru datang biar isinya bisa kita pakai."
try / catch
Jaring Pengaman Error
 "Coba lakuin ini (try). Kalau tiba-tiba meledak/error, tolong tangkap (catch) errornya biar aplikasinya nggak ikut mati total."
Fase 2: Jalan Pintas Si Malas (Shorthands)
=>
Arrow Function
 "Terima input di kiri panah, LALU proses menjadi apa yang di kanan panah."
condition ? A : B
Ternary Operator (If/Else Singkat)
 "Apakah kondisinya benar (?) Kalau YA pilih A. Kalau TIDAK (:) pilih B."
A && B
Logical AND (Hanya Jika)
 "Hanya Jika A benar, tolong kerjakan B. (Biasa dipakai nampilin loading spinner)."
A || B
Logical OR (Alternatif)
 "Coba pakai A. Kalau A kosong/gagal, yaudah pakai B aja."
data?.nama
Optional Chaining
 "Coba cari data 'nama'. Tapi kalau kardus 'data'-nya ternyata kosong, jangan error ya, diemin aja."
Fase 3: Membongkar & Menggandakan Data
.map()
Mesin Fotokopi List
 "Ada 10 data? Tolong copy-paste cetakan HTML-nya 10 kali sesuai jumlah datanya."
.filter()
Saringan Data
 "Pilih dan buang data yang nggak sesuai kriteria (misal: cuma tampilin item yang udah dibayar)."
const { x } = y
Destructuring Object
 "Langsung ambil barang bernama 'x' dari dalam kardus besar 'y'."
[...A, B]
Spread Operator
 "Tumpahkan isi keranjang A, lalu tambahkan barang B di sebelahnya."
Fase 4: Nyawa React / Next.js (The Hooks)
useState()
Buku Catatan Memory
 "Bikin satu memori untuk nyatet perubahan (misal: status tombol dari Off jadi On)."
useEffect()
Alarm / Efek Samping
 "Jalankan fungsi ini OTOMATIS saat halaman web pertama kali terbuka."
props
Oper Barang ke Anak
 "Bos (halaman utama) nitip barang (warna, teks) ke anak buahnya (komponen tombol)."
className=""
Penamaan Baju (CSS)
 "Pakaikan baju desain ini ke elemen HTML tersebut (Kalau di Next.js ini buat Tailwind CSS)."
onClick={}
Tombol Trigger
 "Kalau kotak ini diklik pake mouse, tolong jalankan perintah di dalam kurung kurawal."
onChange={}
Detektor Ketikan
 "Setiap kali user ngetik 1 huruf di kolom input, tolong tangkap hurufnya."
