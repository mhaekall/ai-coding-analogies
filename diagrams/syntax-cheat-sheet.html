Kamus "Bahasa" Mesin

Memahami pola kalimat kode (Syntax) agar tidak bingung saat membaca hasil buatan AI.

1. Ngobrol dengan Pihak Luar (Fetch)

Ini adalah cara kodemu "menelepon" Dapur (Backend) atau layanan lain (seperti OpenAI) untuk meminta atau mengirim sesuatu.

fetch('/api/chat', {
  method: 'POST',
  body: JSON.stringify({ pesan: "Halo" })
})
Analogi Kurir Paket:
• fetch('/api/chat') = "Tolong pergi ke alamat jalan /api/chat"
• method: 'POST' = "Tugasmu adalah MENGIRIM barang (POST), bukan cuma mengambil barang (GET)."
• body: JSON.stringify(...) = "Ini lho kotak paketnya yang sudah dilakban rapat-rapat biar aman selama di jalan."

2. Sabar Menunggu (Async / Await)

Javascript itu sifat aslinya buru-buru. Kalau dia nyuruh orang lain, dia nggak mau nungguin orang itu selesai. await memaksa dia untuk sabar.

async function ambilData() {
  const hasil = await fetch('...');
  tampilkan(hasil);
}
Analogi Pesan Kopi:
• Kalau tanpa await: Kamu pesan kopi ke Barista, lalu detik itu juga kamu sedot gelas kosong di mejamu (karena Baristanya belum selesai bikin). Error!
• Dengan await: Kamu pesan kopi, lalu diam menunggu (await) sampai kopinya ditaruh di meja, baru kamu minum.

3. Cara Singkat Menyuruh (Arrow Function)

Ini cara modern (zaman now) untuk membuat fungsi/perintah, ditandai dengan tanda panah =>.

const sapaUser = (nama) => {
  return "Halo " + nama;
}
Analogi Singkatan Chat:
Daripada nulis panjang-panjang pakai aturan baku (seperti nulis "Yang Terhormat Bapak..."), kita pakai panah => sebagai jalan pintas.
Dibacanya: "Terima data (nama) => lalu ubah jadi tulisan 'Halo nama'".
