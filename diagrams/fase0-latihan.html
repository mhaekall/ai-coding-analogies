Fase 0: Anatomi Web Dasar

Sebelum merakit mesin canggih (Next.js), kita harus tahu cara memegang obeng dan palu. Latihan ini dirancang agar kamu "merasakan" kodenya, bukan sekadar menghafal teori.

🏠

Analogi Utama: Membangun Rumah

Bayangkan kamu sedang membangun sebuah rumah.
• HTML adalah Batu Bata & Tiang. Ia membentuk ruang tamu, kamar, dan pintu. Tanpa ini, rumahnya tidak ada.
• CSS adalah Cat & Dekorasi interior. Ia membuat rumahmu cantik, memberi warna pada dinding, dan menata letak sofa.
• JavaScript adalah Listrik & Saluran Air. Ia membuat lampu bisa dinyalakan saat saklar ditekan, dan air mengalir saat keran dibuka.

LATIHAN 1 : HTML

Menyusun Batu Bata (Struktur)

Tugasmu: Buat kerangka dasar untuk halaman "Profil Diri". Jangan pikirkan warnanya dulu (pasti akan terlihat sangat jelek dan plain). Fokus pada makna/semantic kontennya.

index.html Kerangka Dasar 
<h1>Budi Santoso</h1>
<p>Halo, saya calon AI Developer dari Indonesia.</p>

<h2>Keahlian Saya</h2>
<ul>
  <li>Prompt Engineering</li>
  <li>Belajar AI Tools</li>
</ul>

<button id="btn-kontak">Hubungi Saya</button>
  • Buka VS Code, buat file baru bernama index.html
  • Ketik tanda seru ! lalu tekan Tab. Ini akan otomatis membuat kerangka dasar HTML.
  • Copy kode di atas dan paste di dalam tag <body>.
  • Buka file tersebut di browser (Chrome/Edge). Cek apakah tampilannya muncul (teks hitam, latar putih).
LATIHAN 2 : CSS

Mengecat Rumah (Styling)

Tampilan HTML standar sangat membosankan. Sekarang kita gunakan CSS untuk memberi "cat" dan merapikan letak "perabotan" (layout). Kita akan menggunakan konsep Flexbox (Bayangkan Flexbox seperti barisan kursi yang bisa merenggang atau merapat secara otomatis).

style.css Penataan & Warna 
/* Targetkan seluruh isi rumah (body) */
body {
  background-color: #1a1a1a;
  color: white;
  font-family: sans-serif;
  text-align: center;
  padding-top: 50px;
}

/* Targetkan tombol spesifik */
button {
  background-color: #3b82f6; /* Biru terang */
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 8px; /* Ujung membulat */
  cursor: pointer;
}
  • Buat file style.css di folder yang sama.
  • Sambungkan CSS ke HTML dengan menambahkan <link rel="stylesheet" href="style.css"> di dalam tag <head> HTML-mu.
  • Paste kode CSS di atas ke file style.css.
  • Refresh browser. Lihat bagaimana "rumah" kamu sekarang punya warna dan tombol yang rapi!
LATIHAN 3 : JS

Memasang Saklar Listrik (Interaksi)

Rumahmu sudah berdiri dan dicat, tapi kalau tombol ditekan, tidak terjadi apa-apa (karena listriknya belum dipasang). Kita akan gunakan JavaScript agar tombol tersebut bisa bereaksi saat ditekan.

script.js Logika Interaksi 
// 1. Cari elemen tombolnya di dalam dokumen (DOM)
const tombol = document.getElementById('btn-kontak');

// 2. Beri "Telinga" (Listener) agar ia mendengar jika diklik
tombol.addEventListener('click', () => {
  // 3. Apa yang terjadi saat diklik?
  alert('Terima kasih! Saya akan segera menghubungi Anda.');
  
  // Bonus: Ubah teks tombol secara dinamis
  tombol.innerText = 'Pesan Terkirim ✅';
  tombol.style.backgroundColor = '#10b981'; // Berubah hijau
});
  • Buat file script.js di folder yang sama.
  • Sambungkan ke HTML dengan menambahkan <script src="script.js"></script> tepat sebelum tag penutup </body>.
  • Paste kode JS di atas.
  • Refresh browser. Klik tombol birunya, dan lihat "listrik" (interaksi) bekerja mengubah warna dan memunculkan pop-up!
