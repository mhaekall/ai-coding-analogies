Kamus Besar Syntax Alien 👽

Panduan membaca simbol-simbol aneh (Syntax) di JavaScript, React, & Next.js yang sering di-generate oleh AI.

1. Simbol Logika Singkat (Shortcut)
&&

Tombol "Hanya Jika" (Logical AND)

Daripada nulis aturan panjang if (sedangLoading) { tampilkan_spinner }, programmer menggunakan && agar kode lebih pendek.

{ isLoading && <Spinner /> }
Cara bacanya: "JIKA isLoading itu benar (bernilai true), MAKA tolong tampilkan <Spinner /> di layar. Kalau salah, cuekin aja."
? :

Persimpangan Jalan (Ternary Operator)

Ini adalah bentuk sangat singkat dari If/Else. Menghemat 5 baris kode menjadi cuma 1 baris.

const warna = isError ? 'merah' : 'hijau';
Cara bacanya: "Apakah isError terjadi (?) Kalau IYA, jadikan 'merah'. Kalau TIDAK (:) jadikan 'hijau'."
?.

Pencegah Kiamat (Optional Chaining)

Penyebab utama aplikasi crash/blank putih adalah mencoba membaca data yang ternyata isinya KOSONG. Tanda tanya sebelum titik ini mencegah hal itu.

const namaKota = user?.alamat?.kota;
Analogi Kotak Kosong: "Coba buka data user. ADA isinya? Kalau ada, coba buka alamat. ADA? Kalau ada, ambil kota. Kalau di tengah jalan ternyata KOSONG, udah stop aja, jangan dipaksa ntar error."
2. Mengolah Data (Data Manipulation)
.map()

Mesin Fotokopi (Array Map)

Sangat sering dipakai untuk menampilkan daftar panjang (misal: list produk, history chat, daftar menu) tanpa harus mengetik HTML satu per satu.

{daftarMenu.map((item) => (
  <li> {item.nama} </li>
))}
Cara bacanya: "Tolong ambil toples daftarMenu. Untuk setiap satu item di dalamnya, cetakkan sebuah elemen <li> ke layar." AI pakai ini agar jika menunya ada 100, dia nggak perlu ngetik HTML 100 kali.
{ }

Bongkar Kardus (Destructuring)

Cara cepat mengeluarkan isi dari sebuah objek (kardus besar) tanpa harus menyebut nama kardusnya berkali-kali.

// Cara kuno:
const nama = user.nama;
const umur = user.umur;

// Cara modern (Destructuring):
const { nama, umur } = user;
Cara bacanya: "Dari kardus besar bernama user, tolong langsung bongkar dan ambil barang bernama nama dan umur lalu taruh di meja."
3. Khusus React & Next.js (The Hooks)
useState

Buku Catatan Sementara (State)

Aplikasi butuh ingatan untuk merespon klik user (misal: mencatat angka jumlah like, mencatat teks yang sedang diketik user).

const [jumlah, setJumlah] = useState(0);
Cara bacanya: "Buatkan satu catatan memori bernama jumlah dengan nilai awal 0. Siapkan juga tombol saklar bernama setJumlah. Kalau nanti aku mau ubah angkanya, aku akan panggil tombol setJumlah ini."
useEffect

Alarm / Efek Samping

Menyuruh aplikasi melakukan sesuatu SECARA OTOMATIS saat halaman baru dibuka, atau saat ada data yang berubah.

useEffect(() => {
  fetchDataDariDatabase();
}, []);
Cara bacanya: "Tolong pasang alarm! Begitu halaman web ini selesai di-load di layar HP pengguna, langsung jalankan fungsi fetchDataDariDatabase() secara otomatis (tanpa nunggu di-klik)."
props

Operan Barang (Properties)

Cara sebuah komponen (induk) memberikan data ke komponen di bawahnya (anak).

// Di file Induk.js
<TombolCantik warna="merah" teks="Simpan" />
Analogi Ngasih Tugas: Bos memanggil anak buahnya (TombolCantik) dan membawakan props: "Nih, kamu kerja ya, warnamu kubikin merah, dan teks di perutmu harus tulisan 'Simpan'". Si anak buah akan menerima dan memakainya.
